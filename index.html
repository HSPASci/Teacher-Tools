📎 Embedding a tool in Canvas

Each tool can be embedded directly into any Canvas page as a live, interactive iframe. Here's how:

1
In Canvas, open a page for editing and click the </> HTML editor button (bottom of the toolbar).
2
Click the </> Embed button on any tool above — it copies the ready-made iframe code to your clipboard.
3
Paste the code into the Canvas HTML editor and save. The tool will appear as a fully interactive panel on your Canvas page.
<iframe src="https://YOUR-USERNAME.github.io/YOUR-REPO/sticky-note-board.html" width="100%" height="700" style="border:none;border-radius:10px;"></iframe>
4
Replace YOUR-USERNAME and YOUR-REPO with your GitHub details, and swap the filename for whichever tool you're embedding.
